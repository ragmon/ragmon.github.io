Description

"RC Revolution" is a fun racing game where you drive radio-controlled cars. In the game, you can compete with others on exciting tracks, perform cool stunts, and customize your car. The game is available on Steam and offers both single-player and multiplayer modes. You can choose from various cars and race on different tracks, aiming to be the fastest and perform the best tricks. It’s a great game for fans of racing and RC cars!

Responsibilities

  • Developed the backend architecture of the game using the NestJS framework and TypeScript. My work involved managing game data, handling player interactions, and ensuring smooth server-side operations.
  • Configured continuous integration and continuous deployment (CI/CD) pipelines, enabling seamless updates and automated deployment of the game to Heroku. This streamlined our release process and ensured rapid delivery of new features.
  • Created comprehensive API documentation using OpenAPI and Swagger. This documentation served as a valuable resource for other developers, providing clear guidelines on how to integrate and utilize our backend services.
  • Performed extensive testing and debugging to identify and resolve issues within the backend. My efforts ensured that the backend was robust and worked in harmony with the game's frontend.
  • Worked closely with the Unity development team to coordinate the integration of backend services with the game client. Our collaboration was crucial for achieving seamless gameplay and a cohesive user experience.
  • Conducted in-depth research to identify and select the optimal technologies for the project. This involved evaluating tools and frameworks that would enhance performance and scalability.
  • Set up and managed Docker environments to create consistent development and production setups. This included configuring environments for product deployment and testing, which improved system stability and deployment efficiency.
  • Wrote and implemented bash scripts to automate various development and deployment processes. These scripts significantly increased our workflow efficiency by automating repetitive tasks.
  • Contributed to the research and development of artificial intelligence for the car bots within the game. This involved creating intelligent and realistic bot behavior to enhance the gaming experience.
  • Developed and executed tests using Vitest to ensure code quality and reliability. My testing efforts were crucial in maintaining high standards and minimizing bugs in the codebase.
  • Implemented database seeders to populate the system with test data. This was essential for simulating various scenarios during testing and ensuring the database was prepared for real-world use.
